# Weather App

A modern and responsive Weather App built using HTML, CSS, and JavaScript.

This project was created to practice JavaScript concepts like API integration, asynchronous programming, DOM manipulation, event handling, dynamic UI updates, and responsive frontend design.

## Features

* Real-time weather data using API
* Search weather by city name
* Dynamic temperature updates
* Humidity display
* Wind speed display
* Pressure display
* Screen switching between search and weather view
* Glassmorphism UI design
* Responsive layout
* Hover effects and animations
* Modern weather interface

## Tech Stack

* HTML
* CSS
* JavaScript
* OpenWeatherMap API

## Concepts Practiced

* API Integration
* Async/Await
* Fetch API
* DOM Manipulation
* Event Listeners
* Dynamic UI Updates
* JSON Handling
* Flexbox
* Glassmorphism UI
* Responsive Design

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Add your API key inside `config.js`
4. Open `index.html` in your browser
